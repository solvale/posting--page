**Blog da SOL**

Um site simples para criação e exibição de posts, simulando um blog ou rede social.

# Descrição

Este projeto permite ao usuário criar posts com título e conteúdo, enviando-os para uma API externa (JSONPlaceholder) e exibindo o resultado na página. É uma demonstração de integração front-end com API usando HTML, CSS e JavaScript.

# Funcionalidades

- Formulário para inserir título e conteúdo do post
- Envio do post via fetch para a API
- Exibição do post enviado na página
- Design responsivo com gradiente de fundo

# Tecnologias Utilizadas

- **HTML5**: Estrutura da página
- **CSS3**: Estilização, incluindo gradientes e sombras
- **JavaScript**: Lógica para envio de dados e manipulação do DOM
- **API**: JSONPlaceholder (https://jsonplaceholder.typicode.com/posts)

# Como Executar

1. Clone ou baixe os arquivos do projeto.
2. Abra o arquivo `index.html` em um navegador web.
3. Preencha o formulário com título e conteúdo.
4. Clique em "Postar" para enviar e visualizar o post.

# Estrutura do Projeto

- `index.html`: Página principal
- `style.css`: Arquivo de estilos
- `main.js`: Lógica JavaScript
